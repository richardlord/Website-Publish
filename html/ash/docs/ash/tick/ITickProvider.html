Packageash.tick
Interfacepublic interface ITickProvider
Implementors FixedTickProvider, FrameTickProvider, StarlingFixedTickProvider, StarlingFrameTickProvider

The interface for a tick provider. A tick provider dispatches a regular update tick to act as the heartbeat for the engine. It has methods to start and stop the tick and to add and remove listeners for the tick.
