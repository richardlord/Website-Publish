Packageash.integration.swiftsuspenders
Classpublic class SwiftSuspendersEngine
InheritanceSwiftSuspendersEngine Engine Object

A custom engine class for games that use SwiftSuspenders for dependency injection. Pass a SwiftSuspenders injector to the constructor, and the engine will automatically apply injection rules to the systems when they are added to the engine.
