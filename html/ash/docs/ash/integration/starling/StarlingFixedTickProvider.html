Classpublic class StarlingFixedTickProvider
InheritanceStarlingFixedTickProvider Signal1 SignalBase Object
Implements ITickProvider, starling.animation.IAnimatable

Uses a Starling juggler to provide a frame tick with a fixed frame duration. This tick ignores the length of the frame and dispatches the same time period for each tick.

timeAdjustmentproperty 
public var timeAdjustment:Number = 1

Applies a time adjustement factor to the tick, so you can slow down or speed up the entire engine. The update tick time is multiplied by this value, so a value of 1 will run the engine at the normal rate.
