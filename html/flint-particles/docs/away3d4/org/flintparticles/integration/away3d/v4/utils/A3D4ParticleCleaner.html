Packageorg.flintparticles.integration.away3d.v4.utils
Classpublic class A3D4ParticleCleaner
InheritanceA3D4ParticleCleaner Object

Away3D 4 objects and materials should be disposed of, by calling their dispose method, when they are no longer required. This class will call the dispose method on the image object of particles when they die. To use it, create an instance of the class and add any Emitters to it that use Away3D 4 objects. All particles on that emitter will be cleaned up when they die.
