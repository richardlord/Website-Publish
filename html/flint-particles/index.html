About

Flint Particles is an open-source particle system engine created for Flash in 2008 by Richard Lord. It has a mudular structure designed for flexibility and extensibility and core code that is optimised for high performance.

Getting Started

  1. Grab the source code, documentation and examples for the final version of Flint. Either grab the current downloads of each or check out the code from the GitHub repository.
  2. Work through the beginners tutorial to get a feel for how Flint works.
  3. Look at some of the examples, they all have full source code for Flash and for Flex.
  4. Check out the documentation to learn more about Flint.
