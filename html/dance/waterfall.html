Created in 2002, Waterfall is an innovative interactive dance production for use on computers. Suitable for all ages it combines live action and user control to let the user interact with and influence the dance as it takes place.

 

Waterfall is inspired by water – water in nature, in tears, as an object for play, and as a source for life. Each of the five main sections of the work is inspired by a different aspect of water and is interactive in a different way. The whole dance takes about an hour to complete. There is no structured interface - you use the mouse to explore the environment and see how the dance and dancers respond to you. And if you get confused, there's a help system to guide you along.

 

Waterfall was released as a CD-ROM for Windows and Mac in 2002.
