ABOUT

Created in 2002, Waterfall is an experimental interactive dance production. Suitable for all ages it combines contemporary dance with elements of gameplay to let the user play with and influence the dance.

 

Waterfall is inspired by water – water in nature, in tears, as an object for play, and as a source for life. Each of the five main sections of the work is inspired by a different aspect of water and is interactive in a different way. The whole dance takes about an hour to complete. There is no structured interface - use the mouse to explore the environment and see how the dance and dancers respond to you.

 

Waterfall was released as a CD-ROM for Windows and Mac in 2002 and unfortunately is not compatible with current computer operating systems.

Richard Lord - Writer, Choreographer, Game Developer - www.richardlord.net
