Robotlegs example project with source

Posted on 04 March 2007

Any comparison of frameworks wouldn't be complete without Robotlegs. I included Robotlegs in my session at the London Flash Platform User Group recently, but didn't post the example project here because the framework was in a state of flux. Robotlegs is now settling down as it approaches its imminent 1.0 release, and the MVCS implementation in it is unlikely to change further, so here's my example.

For this Robotlegs example I've used exactly the same project as in the previous examples for other frameworks. Robotlegs is not prescriptive about your application's architecture, but it does include a default MVCS implementation for those that wish to use it. I've used that default implementation here.
